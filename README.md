## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/heidesebastiao95/CMS-Panel.git
   ```
2. Create Database
   ```sh
    php artisan db create database blog_register
   ```
3. And run the initial migrations
   ```js
   php artisan migrate
   ```
